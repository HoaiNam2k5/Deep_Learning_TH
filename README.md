# Deep Learning - Thực hành (TH)

Repository này lưu bài thực hành môn Deep Learning, hiện tại gồm notebook về **NumPy** và **Pandas**.

## Nội dung hiện có

- `Lab1_Numpy_Pandas.ipynb`: Bài lab luyện tập thao tác mảng và xử lý dữ liệu cơ bản, ví dụ:
  - Tạo mảng ngẫu nhiên, kiểm tra điều kiện với `all`, `any`
  - So sánh mảng với `greater`, `less`, `greater_equal`, `less_equal`
  - Tạo các mảng đặc biệt (`zeros`, `ones`, `identity`, ...)
  - Các bài tập xử lý dữ liệu nền tảng

## Yêu cầu môi trường

- Python 3.x
- Jupyter Notebook hoặc Google Colab
- Thư viện:
  - `numpy`
  - `pandas`

## Cách sử dụng

### 1) Chạy trên máy local

1. Clone repository:
   ```bash
   git clone https://github.com/HoaiNam2k5/Deep_Learning_TH.git
   cd Deep_Learning_TH
   ```
2. Cài thư viện cần thiết:
   ```bash
   pip install numpy pandas notebook
   ```
3. Mở notebook:
   ```bash
   jupyter notebook
   ```
4. Chọn file `Lab1_Numpy_Pandas.ipynb` để chạy.

### 2) Chạy trên Google Colab

- Mở trực tiếp notebook qua badge Colab ở đầu file notebook, hoặc tải file `.ipynb` lên Colab.
